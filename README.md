# NightModePlusScreen
NightModePlusScreen est un script AutoHotKey qui vous permet de réduire encore davantage la luminosité perçue de l'écran de votre PC — même lorsqu'elle est déjà réglée au minimum — afin de protéger vos yeux. Pour ce faire, il modifie directement les paramètres de couleur de votre écran.

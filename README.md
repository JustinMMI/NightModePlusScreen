# NightModePlusScreen
NightModePlusScreen est un script AutoHotKey qui vous permet de réduire encore davantage la luminosité perçue de l'écran de votre PC — même lorsqu'elle est déjà réglée au minimum — afin de protéger vos yeux. Pour ce faire, il modifie directement les paramètres de couleur de votre écran.

Utilisation :
- Appuyez sur Ctrl + Alt + N pour activer le mode nuit
- Appuyez sur Ctrl + Alt + J pour désactiver le mode nuit

# English
NightModePlusScreen is an AutoHotKey script that allows you to further reduce the perceived brightness of your PC screen—even when it’s already set to the minimum—to protect your eyes. It does this by changing your screen's color parameters directly.

How to use:
- Press Ctrl + Alt + N to enable night mode
- Press Ctrl + Alt + J to disable night mode
